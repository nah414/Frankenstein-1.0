## ğŸ”§ Troubleshooting

### "customtkinter not found"
```powershell
pip install customtkinter
```

### "psutil not found"
```powershell
pip install psutil
```

### Widget doesn't appear
1. Check Python version: `python --version` (need 3.10+)
2. Check tkinter: `python -c "import tkinter; print('OK')"`
3. Try CLI mode: `python frankenstein.py --cli`

### Icon doesn't show
1. Run icon generator: `python scripts\generate_icon.py`
2. Re-run shortcut creator: `powershell -ExecutionPolicy Bypass -File scripts\create_desktop_shortcut.ps1`
